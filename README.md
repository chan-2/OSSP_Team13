# OSSP_Team13

## App Name
LUVWAY

## How to use
#### 매일 반복되는 데이트 코스로 인해 지루함을 느끼는 연인들에게 **랜덤**으로 데이트할 *지하철역*을 추천해줍니다.



1. 사용자에게 기본으로 5개의 코인이 주어집니다. 
2. LUWAY 페이지에 접속하여 하트가 그려진 카드를 클릭합니다.
- 새로운 카드를 뒤집을 때마다 코인이 하나씩 줄어드니 조심해주세요!
3. 33개의 지하철역 중 하나의 역이 랜덤으로 나오게 됩니다.
4. 해쉬태그를 통해 지하철역 근처 놀거리, 구경거리를 볼 수 있습니다.
5. 그날의 날씨도 함께 띄워지니 적절한 옷차림과 함께 데이트를 즐겨주시면 됩니다.
